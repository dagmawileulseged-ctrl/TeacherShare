# TeacherShare Frontend (starter)

This is a minimal Next.js + TypeScript + Tailwind starter scaffold for the TeacherShare frontend.

Quick start:

```bash
npm install
npm run dev
```

Files added: basic pages, `Header` and `TopicForm` components, and Tailwind setup.
